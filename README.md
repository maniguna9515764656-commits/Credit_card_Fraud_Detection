# Credit Card Fraud Detection

## Project Overview
This project is a Credit Card Fraud Detection System developed using Python. The system identifies suspicious credit card transactions based on predefined rules such as transaction amount, transaction country, and transaction time.

## Objectives
- Detect fraudulent transactions.
- Analyze transaction risk factors.
- Generate transaction reports.
- Provide a simple fraud detection mechanism without machine learning.

## Features
- Transaction Amount Analysis
- Country Verification
- Transaction Time Verification
- Risk Score Calculation
- Fraud Detection Report
- Risk Analysis Graph

## Technologies Used
- Python
- Pandas
- Matplotlib

## Project Workflow
1. Load transaction details.
2. Analyze transaction amount.
3. Verify country information.
4. Check transaction time.
5. Calculate risk score.
6. Classify transaction as:
   - Safe
   - Suspicious
   - Fraud
7. Generate final report.

## Sample Output

Transaction Amount : 90000

Country : USA

Transaction Hour : 2

Risk Score : 5

Status : FRAUD

## Conclusion
This project demonstrates how credit card fraud can be detected using rule-based logic in Python without using machine learning algorithms.
